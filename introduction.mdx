      Get up and running with Africa's Talking APIs and Google Gemini AI in minutes
    </p>
  </div>

  <Steps>
    <Step title="Clone the repository and set up your environment">
      Clone the course repository and create a virtual environment for Python dependencies.

      ```bash
      git clone https://github.com/DaggieBlanqx/Google-Africastalking-AI-Course
      cd Google-Africastalking-AI-Course
      python -m venv venv
      source venv/bin/activate  # On Windows: venv\Scripts\activate
      ```
    </Step>

    <Step title="Install dependencies">
      Install all required Python packages using the Makefile command.

      ```bash
      make install
      ```

      This installs Flask, the Africa's Talking SDK, Google AI libraries, and other dependencies.
    </Step>

    <Step title="Configure your credentials">
      Create a `.env` file in the project root with your Africa's Talking and Google Cloud credentials.

      ```bash
      AT_USERNAME=your_at_username
      AT_API_KEY=your_at_api_key
      AT_VOICE_NUMBER=your_voice_enabled_phone_number
      AT_SHORTCODE=8995
      PROJECT_ID=your_gcp_project_id
      GEMINI_API_KEY=your_gemini_api_key
      ```

      <Note>
        Get your Africa's Talking credentials at [account.africastalking.com](https://account.africastalking.com/) and your Gemini API key at [aistudio.google.com](https://aistudio.google.com/api-keys).
      </Note>
    </Step>

    <Step title="Set up tunneling and start the server">
      Expose your local server to the internet using ngrok or another tunneling tool, then start the Flask application.

      ```bash
      # In one terminal, start ngrok
      ngrok http 5000

      # In another terminal, start the Flask app
      make dev
      ```

      Your server is now running and accessible via the ngrok URL. Configure this URL in your Africa's Talking app settings for webhook callbacks.
    </Step>
  </Steps>
